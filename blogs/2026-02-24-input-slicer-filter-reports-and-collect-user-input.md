Previously known as the "text slicer" while in preview, this visual lets you type or paste values directly into a slicer to filter your report—no scrolling or searching through long lists required. What you can do with the input slicer Report consumers can quickly filter the report to what they are looking for using the input slicer. If you've ever needed to filter a report by a specific order ID, customer name, or product code, the input slicer is designed for exactly that.
